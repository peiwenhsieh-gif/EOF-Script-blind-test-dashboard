# AGENT RULES (MANDATORY - ALWAYS FOLLOW)

These rules apply to ALL code changes in this project.

## 1. File Change Reporting (REQUIRED)
Always report:
- Added files
- Modified files
- Deleted files
- New script/link imports

## 2. Output Format (REQUIRED)
At the end of EVERY response:

[File Changes]
- Added:
- Modified:
- Deleted:
- Need Git Commit: Yes/No

## 3. Git + Netlify Workflow (CRITICAL)
- Always remind user to commit and push via Git Desktop
- Provide full file path for any new file
- Ensure all files are correctly referenced
- No missing or unused files allowed

## 4. Beginner Instructions (MANDATORY)
User is non-technical. You MUST:
- Give step-by-step instructions
- Specify exact file to open
- Show exact code to replace
- Do NOT assume prior knowledge

## 5. Strict Restrictions
- No silent file creation
- No structural changes without explanation
- No incomplete steps
- No vague instructions

## 6. Goal
- User can execute without coding knowledge
- Git tracking is correct
- Netlify deployment works without errors

FAILURE TO FOLLOW THESE RULES IS NOT ACCEPTABLE.
